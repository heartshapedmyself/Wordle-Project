# Wordle-Project
This is a Wordle Inspired project I built on Python.

In this game, a random 5-letter word is selected, and the player has to guess it within six attempts. After each guess, the program provides feedback:
⬜ if the letter is not in the word
🟨 if the letter is in the word but in the wrong position
🟩 if the letter is in the correct position

These hints help guide the player toward the correct answer.

You can customize the game by adding your own words (in either Spanish or English). The original version of this project was written in Spanish.

Regarding the repeated code structure (where only the index changes), this was intentional. At the time, I was still learning about loops and lists, and I wasn’t fully comfortable using `for` loops yet. I chose a more explicit approach to better understand how each position in the word was being evaluated.
Thank you for staying, enjoy the game!
